# ai

Write in natural language what you want to run.

## Usage

```shell
nx ai [prompt]
```

Install `nx` globally to invoke the command directly using `nx`, or use `npx nx`, `yarn nx`, or `pnpm nx`.

## Options

### help

Type: `boolean`

Show help

### prompt

Type: `string`

What command you want to run.

### version

Type: `boolean`

Show version number
